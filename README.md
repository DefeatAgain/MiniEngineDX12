# MiniEngineDX12
一个移植并重新封装的渲染器，目前还在开发中...<br/>
+ 支持多线程渲染
+ 简单的内存管理，描述符管理
+ 重新封装Device层，便于后续开发
+ 目前实现:
  + PBR
  + CSM, MSAA Shadow
